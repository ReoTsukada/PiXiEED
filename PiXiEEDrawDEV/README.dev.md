# PiXiEEDrawDEV

This directory is a local development workspace for continuing the PiXiEEDraw file-split work.

- Production/stable PiXiEEDDraw lives in `../pixiedraw/`.
- This directory keeps the in-progress split `assets/js/app.js` and `assets/js/modules/...` files.
- This directory is ignored by Git and should not be committed as part of the production site.
- Use this workspace for split verification before copying reviewed changes back into `../pixiedraw/`.
