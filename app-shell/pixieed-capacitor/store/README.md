# Store Submission Templates

PiXiEED を Google Play / App Store へ提出する時に使う文言テンプレートです。

- Release build の基本コマンド:
  - `npm run android:build:aab`
  - `npm run ios:archive`
  - `npm run ios:export:appstore`
- Android 署名設定:
  - `android/keystore.properties.example` をコピーして `android/keystore.properties` を作成
- iOS export 設定:
  - `ios/export-options/app-store-connect.plist`

- [checklist.md](./checklist.md): 提出前チェック
- [google-play-ja.md](./google-play-ja.md): Google Play 日本語文言
- [app-store-ja.md](./app-store-ja.md): App Store 日本語文言
- [app-store-en.md](./app-store-en.md): App Store 英語文言

現時点の基本 URL:

- Privacy: `https://pixieed.jp/privacy/`
- Account deletion: `https://pixieed.jp/account-deletion/`
- Terms: `https://pixieed.jp/terms/`
- Support: `https://pixieed.jp/contact/`
