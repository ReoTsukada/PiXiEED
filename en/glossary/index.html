Basic Terms

Foundation vocabulary for pixel art

Pixel ピクセル
The smallest visual unit of a bitmap image.
Dot ドット
Common alternate name for a pixel in pixel-art communities.
Pixel Art ピクセルアート
Art made by intentionally designing visible pixels.
Dot Art ドット絵
Japanese term often used for pixel art.
Sprite スプライト
2D image used for characters, items, or effects in games.
Tile タイル
A small reusable image block used to build environments.
Tileset タイルセット
A sheet containing many tile variants for map building.
Sprite Sheet スプライトシート
A single image containing multiple sprite frames.
Bitmap ビットマップ
Image format represented by a grid of pixels.
Raster Image ラスター画像
General term for pixel-based images.
Indexed Color インデックスカラー
Color mode where pixels store palette indices instead of full RGB values.
Palette パレット
The set of colors used in an artwork.
Palette Limit パレット制限
Rule restricting the number of usable colors.
Color Palette カラーパレット
A curated color set for consistent visual style.
RGB RGB
Color model based on red, green, and blue channels.
Indexed Color (EN) Indexed Color
English notation for palette-index based color workflow.

Resolution and Size

Canvas dimensions and scaling behavior

16x16 16×16
Very small size used for simple icons or tiny sprites.
32x32 32×32
Common small size for characters and tile elements.
64x64 64×64
Mid-size sprite canvas with more detail capacity.
128x128 128×128
Larger canvas suited for more complex pixel pieces.
256x256 256×256
High-detail pixel canvas often used for backgrounds.
Low Resolution 低解像度
Image with fewer pixels and stronger blocky character.
Pixel Perfect ピクセルパーフェクト
Rendering state where pixel edges stay crisp and aligned.
Native Resolution ネイティブ解像度
The original intended resolution of an asset.
Scaling スケーリング
Resizing an image to a different display size.
Nearest Neighbor ニアレストネイバー
Scaling algorithm that preserves hard pixel edges.
Upscale アップスケール
Enlarging image dimensions from the source size.
Downsample ダウンサンプル
Reducing image dimensions, often losing detail.

Creation and Collaboration

Terms used in practical PiXiEEDraw workflows

Multi-layer マルチレイヤー
Separating lines, colors, and effects into editable layers.
Collaboration 共同編集
Multiple creators editing one project together.
Realtime Drawing リアルタイム描画
Near-instant sync of drawing updates to participants.
Frame Animation フレームアニメーション
Animation built by switching frame-by-frame drawings.
Tilemap タイルマップ
Map built by placing tile IDs on a grid.
Sprite Authoring スプライト作成
Workflow of drawing, organizing, and exporting sprites.
Timelapse タイムラプス
Playback of drawing history over time.

Rendering and Shading

Visual quality and lighting terminology

Outline アウトライン
Outer contour line that clarifies shape readability.
Inline インライン
Interior line used for separation or detail accents.
Cel Shading セルシェーディング
Stepped shading style with hard light-shadow transitions.
Shading シェーディング
Adding light and shadow to create volume.
Highlight ハイライト
Brightest area where light directly hits a surface.
Shadow シャドウ
Darker area blocked from direct light.
Hard Shadow ハードシャドウ
Sharp-edged shadow with strong boundary definition.
Soft Shadow ソフトシャドウ
Blurred or gradual shadow edge.
Light Source ライトソース
Origin point or direction of light in a scene.
Lighting Source 光源
Japanese equivalent of light source.
Contrast コントラスト
Difference between light and dark values.
High Contrast ハイコントラスト
Strong value difference for high readability.
Dithering ディザリング
Patterned pixel mixing to simulate intermediate tones.
Checker Dither チェッカーディザ
Dither pattern using checkerboard placement.
Pattern Dither パターンディザ
Custom repeating dither pattern for texture or gradients.
Banding バンディング
Unwanted parallel pixel runs that flatten forms.
Pillow Banding ピローバンディング
Concentric shading around forms that causes unnatural puffiness.
Cluster クラスター
Connected group of same-color pixels treated as a shape unit.
Anti-aliasing アンチエイリアス
Smoothing jagged edges with transitional pixels.
Selective Anti-aliasing セレクティブアンチエイリアス
Applying anti-aliasing only where it helps readability.
Subpixel サブピクセル
Fine perceived positioning beyond strict full-pixel movement.
Pixel Density ドット密度
How concentrated detail is within an area.
Noise ノイズ
Unintentional pixel variation that harms clarity.
Pixel Cluster ピクセルクラスター
Cluster-level analysis of pixel grouping quality.

Animation Terms

Frame timing and motion structure

Frame フレーム
Single image unit in an animation sequence.
Frame Rate フレームレート
Number of frames shown per second (fps).
Loop ループ
Animation that repeats continuously from end to start.
Keyframe キーフレーム
Main pose frame that defines major motion states.
In-between インビトゥイーン
Intermediate frame between keyframes.
Tiling Animation タイリングアニメ
Looped tile-based animation for water, lava, or flow effects.
Sprite Animation スプライトアニメーション
Animation by switching sprite frames over time.
Sheet Animation シートアニメーション
Playback using frame regions on one sprite sheet.
Timeline タイムライン
UI track for frame order and timing control.
Frame Step フレームステップ
One-frame forward/backward navigation for checking motion.

Sprite and gameplay-oriented asset vocabulary

Character Sprite キャラクタースプライト
Sprite representing a playable or main character.
Enemy Sprite 敵スプライト
Sprite used for hostile entities.
NPC NPC
Non-player character controlled by game logic.
Tilemap タイルマップ
Grid-based map assembled from tile IDs.
Autotile オートタイル
Tile system that automatically selects edge/corner variants.
Walk Animation ウォークアニメーション
Frame set used for walking movement.
Attack Animation 攻撃アニメーション
Frame set for attack motions and strikes.
Effect Sprite エフェクトスプライト
Sprite for particles, impact, magic, smoke, and VFX.
UI Sprite UIスプライト
Sprite element used in interface components.
Icon アイコン
Small symbolic image for actions or states.

Color restrictions and retro palette usage

Limited Palette 限定パレット

NES Palette NESパレット
Color set inspired by NES-era hardware constraints.
Game Boy Palette GameBoyパレット
Palette inspired by classic Game Boy tonal style.
SNES Palette SNESパレット
Palette influenced by SNES-era color aesthetics.
Retro Palette レトロパレット
Palette designed to emulate old-system color feeling.
Color Limit 色数制限
Upper bound on how many colors can be used.
4-color Limit 4色制限
Constraint allowing only four colors in one piece.
16-color Limit 16色制限
Classic restriction used in many retro workflows.
Color Variation カラーバリエーション
Alternative color versions of the same base sprite.
Palette Swap パレットスワップ
Changing indexed colors to produce variants efficiently.
Palette Animation パレットアニメーション
Animating by cycling palette colors over time.

Background and Tilemap

Terms for map construction and scrolling layers

Background Tile 背景タイル
Reusable tile specifically for background surfaces.
Tilemap タイルマップ
Structured map data made from tile placement.
Layer レイヤー
Separate depth or composition plane in rendering.
Parallax パララックス
Depth illusion by moving layers at different speeds.
Parallax Scroll パララックススクロール
Scrolling implementation of the parallax effect.
Map Chip マップチップ
Japanese term for tile assets used in map editors.
Chunk チャンク
Larger grouped tile block for map management.
Grid グリッド
Underlying alignment lattice for placement and editing.
Tile Grid タイルグリッド
Grid aligned to tile size and tile boundaries.
Seamless Tile シームレスタイル
Tile that repeats without visible seams.

Production Pipeline

Typical workflow from concept to final export

Rough ラフ
Early sketch stage for composition and proportion decisions.
Blocking ブロッキング
Placing large masses before refining details.
Silhouette シルエット
Outline readability check independent of inner details.
Shape シェイプ
Primary form design and contour logic.
Cleanup クリーンアップ
Removing noise and correcting lines/forms.
Palette Cleanup パレット整理
Merging redundant colors and clarifying palette roles.
Color Pass カラーパス
Stage focused on color and value correction.
Refine リファイン
Final detail tuning and readability optimization.
Final Polish 最終仕上げ
Last pass before export and publication.

Editor UI and Tool Terms

Operational vocabulary used in pixel editors

Pixel Brush ピクセルブラシ
Brush tool intended for pixel-by-pixel painting.
Dot Pen ドットペン
Pen tool optimized for single-pixel editing.
Fill 塗りつぶし
Tool that paints connected regions at once.
Line Tool ラインツール
Tool for drawing straight lines between two points.
Bucket Tool バケツツール
Common name for flood fill operation.
Eyedropper スポイト
Tool that samples color from existing pixels.
Grid Display グリッド表示
Overlay showing pixel cell boundaries.
Grid Snap グリッドスナップ
Snaps movement or drawing to grid positions.
Pixel Snap ピクセルスナップ
Forces coordinates to whole pixels to avoid blur.
Mirror Drawing ミラー描画
Reflects strokes across one axis.
Symmetry Drawing 対称描画
Applies strokes across one or more symmetry axes.
Onion Skin オニオンスキン
Shows adjacent frames transparently during animation editing.
Layer Compositing レイヤー合成
Combining visual output of multiple layers.
