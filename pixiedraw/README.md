# PiXiEED Labo

PiXiEEDraw をブラウザから直接操作できるピクセルアートエディターです。ローカル自動保存に対応しており、PC/モバイルどちらでも同じ操作感を提供します。

## 使い方メモ

- 右パネルの「自動保存ファイルを設定」ボタンから保存先ファイルを指定すると、以後は同じファイルにドキュメント全体が自動保存されます。対応ブラウザ（Chrome / Edge / Safari 17.4+ など）では、再読み込み後も同じファイルが復元されます。
- 作業途中の JSON ドキュメントは右パネルの「ファイルを開く」からいつでも読み込めます。同じファイルを選択すれば自動保存の対象にもなります。
- 自動保存ファイルは JSON 形式で書き出されるため、バージョン管理やバックアップも容易です。
- 設定パネルの「エクスポート」ボタンから、PNG / JPEG / SVG / グリッド分割PNG / アニメーションGIF / タイムラプスGIF を選んで書き出せます。
- タイムライン下部には巻き戻し/再生/停止/先送りのコントロールをまとめたボタン群があり、fps 入力の横で 1 フレーム当たりのミリ秒表示を確認しつつ `全体適用` で全フレームへ一括反映できます。
- 右側の「拡張」パネルでは、マルチ中でも端末内だけで動くローカル拡張を使えます。拡張は見える sandbox iframe の中で実行され、独自UI・独自canvas・ローカル音などを作れます。既存のツール欄やメインキャンバスには触れず、共有状態を書き換えるAPIも公開していません。拡張コードからの外部通信も無効化しています。GPT連携は廃止済みで、旧AI設定は起動時にローカルから削除されます。
- 拡張パネルは `有効化` と `拡張ファイルを作る / 既存ファイルを選ぶ` の最小構成です。`拡張ファイルを作る` では、最初から使える `8-Way Sprite Assistant` のテンプレを生成します。N / E / S / W の 4方向だけを編集し、NE / SE / SW / NW は隣接方向どうしの自動 in-between で補完する実用寄りの雛形です。作成または選択した `.js` を VSCode や GPT連携メモ帳で保存すると、自動で再読込されます。拡張パネルの shell は最初から用意されており、拡張コードは `api.ui.getRoot()/clear()/mount()/addStyles()` で中身だけを組めます。ローカル保存は `api.storage.get/set/remove()` を使います。ファイルシステムAPI非対応環境では、既存の `.js` を選び直して更新します。
- GPT にテンプレを編集させる時の前提は、テンプレ先頭コメントと [docs/local-extension-gpt-guide.md](/Users/tsukadareine/Documents/GitHub/PiXiEED/pixiedraw/docs/local-extension-gpt-guide.md) にまとめています。作成した `.js` とこの説明書を一緒に渡す想定です。
