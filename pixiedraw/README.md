# PiXiEED Labo

PiXiEEDraw をブラウザから直接操作できるピクセルアートエディターです。ローカル自動保存に対応しており、PC/モバイルどちらでも同じ操作感を提供します。

## 使い方メモ

- 右パネルの「自動保存ファイルを設定」ボタンから保存先ファイルを指定すると、以後は同じファイルにドキュメント全体が自動保存されます。対応ブラウザ（Chrome / Edge / Safari 17.4+ など）では、再読み込み後も同じファイルが復元されます。
- 作業途中の JSON ドキュメントは右パネルの「ファイルを開く」からいつでも読み込めます。同じファイルを選択すれば自動保存の対象にもなります。
- 自動保存ファイルは JSON 形式で書き出されるため、バージョン管理やバックアップも容易です。
- 設定パネルの「エクスポート」ボタンから、PNG / JPEG / SVG / グリッド分割PNG / アニメーションGIF / タイムラプスGIF を選んで書き出せます。
- タイムライン下部には巻き戻し/再生/停止/先送りのコントロールをまとめたボタン群があり、fps 入力の横で 1 フレーム当たりのミリ秒表示を確認しつつ `全体適用` で全フレームへ一括反映できます。
- 右側の「拡張」パネルでは、マルチ中でも端末内 sandbox で動く拡張機能を使えます。拡張は見える sandbox iframe の中で実行され、独自UI・独自canvas・ローカル音などを作れます。同じ `.js` を共有すれば他の人も同じ機能を再現できます。公開 API を通して既存のツール欄やキャンバス入力へ参加させることもできますが、共有状態の直接変更や外部通信、Supabase など課金系 API への接続は無効化しています。GPT連携は廃止済みで、旧AI設定は起動時にローカルから削除されます。
- 拡張パネルは、保存済み拡張をグリッドで並べる一覧です。`拡張ファイルを作る / 既存ファイルを選ぶ` で拡張を追加し、各カードを `ON / OFF` 切替できます。`拡張ファイルを作る` では、最初から使える `Host API Starter` テンプレを生成します。既存ツール欄への追加、キャンバス入力取得、オーバーレイ描画、ローカル保存まで入った出発点です。テンプレ先頭には `Name / Version / Author / Description` のメタ情報ヘッダも入れてあり、カード上で情報を確認できます。選択中拡張の UI は下部ワークスペースに表示されます。ローカル保存は `api.storage.get/set/remove()` を使います。
- GPT にテンプレを編集させる時の前提は、テンプレ先頭コメントと `pixiedraw/docs/local-extension-gpt-guide.md` にまとめています。作成した `.js` とこの説明書を一緒に渡す想定です。
- host API の仕様は `pixiedraw/docs/extension-host-api.md`、最小サンプルは `pixiedraw/docs/extension-host-api-sample.js` に置いてあります。
