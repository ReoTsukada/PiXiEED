PiXiEEDraw Lite（ピクドラ）

16×16のキャンバスで、シンプルにドットを描ける軽量ツールです。ジェリンのプレビューも用意しつつ、PiXiEEDrawとは別実装のライト版としてブラウザですぐ動きます。

ブラウザ上ですぐ描ける

キャンバスは16×16pxで固定（8/16/32px切り替えも可能）。描画・消しゴム・グリッド・PNG書き出しに絞り、専用ページで軽快に動作します。

PiXiEEDraw Liteを開く
  1. 専用ページでキャンバス上をクリック/ドラッグして描画。ガイド線は常時オンです。
  2. パレットかカラーピッカーで色を選び、右クリックまたはAlt/Optionで消しゴムに切り替えられます。
  3. 「PNGを書き出す」で640pxに拡大書き出し。ジェリンの下絵を表示して上から塗ることもできます。

プリセット仕様

キャンバス
16×16 px 固定（8/16/32切替可）/ グリッド常時表示
エンジン
PiXiEEDraw Lite（シンプル描画・ジェリンプレビュー）
書き出し
PNG（640px拡大書き出し）
利用推奨
Chrome / Safari / Edge（ブラウザ版・インストール不要）

ブラウザの自動保存には対応していません。必要に応じてPNGを書き出して保存してください。
