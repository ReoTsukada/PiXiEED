overview

概要

魔王様!!いつまでよければいいですか!? は、敵を避け続けるシンプルなブラウザアクションです。

ルール説明を長く読ませず、すぐ遊ばせることを重視しています。

イベント、配信、SNS共有にも向く短時間ゲームです。

features

主な特徴

ルールが短い

避けるだけなので初見でも理解しやすいです。

すぐ始まる

ブラウザで開いてそのまま遊べます。

短時間で終わる

空き時間やイベント導線に向きます。

共有しやすい

見た目の分かりやすさでSNS導線にも向きます。

scenes

利用シーン

  • 短時間で遊びたい時
  • イベントや配信で見せたい時
  • アクション系を軽く試したい時

how to use

使い方

  1. ゲームを開く
  2. 操作を確認する
  3. 敵を避け続ける
  4. スコアを見て再挑戦する
