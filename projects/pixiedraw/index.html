5 minute guide

PiXiEEDrawの使い方

まずは、開いて、描いて、動かして、保存する流れだけを押さえます。

PiXiEEDrawを開く 手順を見る

quick steps

この4つだけ覚える

1. 開く

ホームや下部メニューからPiXiEEDrawを開きます。最初はそのまま小さく描き始めれば大丈夫です。

2. 描く

ペン、消しゴム、塗りつぶしなどを選び、キャンバスをタップまたはドラッグして描きます。

3. 動かす

フレームを追加するとアニメーションを作れます。まずは2枚の絵を切り替えるだけで動きになります。

4. 保存する

完成したらPNGやGIFで書き出します。投稿したい場合は広場へ進みます。

first try

最初に作るなら

  1. 16x16くらいの小さな絵にする
  2. 色は3〜5色に絞る
  3. 輪郭を描いてから中を塗る
  4. 動かす場合は1か所だけ変える
  5. PNGで保存して確認する
