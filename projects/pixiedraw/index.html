PiXiEEDraw

ブラウザ単体で編集から書き出しまで完結するドット絵エディターです。リアルタイム共有モードでみんなで同時に描けて、フレームアニメーション制作まで1つの画面で完結します。

レイヤー合成・オニオンスキン・カスタムブラシ・自動保存・GIF/PNG出力に対応。PC・タブレット・スマホのいずれでも同じ制作フローで使えます。

開発ノートまとめへ

使い方

  1. 起動後のスタート画面で「新規作成」または「ファイルを開く」を選び、キャンバスサイズや読み込むデータを設定します。
  2. 左レールでツール、右レールでレイヤーやアニメーションを切り替えながらドットを配置します。モバイルでは画面下部のタブから同じ機能にアクセスできます。
  3. 編集中は数秒ごとに自動保存が行われます。エクスポート時はPNG・スプライトシート・タイルセットなど目的に合わせた書き出し形式を選択してください。
  4. 設定メニューから仮想カーソルやUIスケールを端末に合わせて調整できます。モバイルではボタンサイズを変えて描きやすさを確保してください。

Chromeでの利用を推奨しています。Chrome以外のブラウザでは一部機能に制限がかかる場合があります。

PiXiEEDraw の特徴

「ドット絵を描く」「アニメにする」「みんなで仕上げる」をブラウザだけで実現できる構成を目指して開発しています。特に リアルタイム共同編集 と アニメーション制作 を強化しています。

  • リアルタイム共同編集（マルチ） プロジェクトキーを使って参加者を接続し、同じ作品を同時進行で制作できます。マスター/参加者の役割とセル割当で分業しやすい設計です（テスト運用中）。
  • アニメーション制作対応 フレーム編集、FPS設定、オニオンスキン表示を使いながら動きの確認が可能。完成後はGIFとしてそのまま書き出せます。
  • モバイル最適 UI スマホの縦画面・横画面をどちらも想定したレスポンシブ設計。指での描画やペン操作でもメニューが邪魔にならず、ステージとツールの並びを自動で再配置します。
  • 自動保存 & 復帰サポート 描いている最中でも数秒ごとにローカルへスナップショットを保存。ブラウザを閉じても直前の状態を復元でき、保存忘れのストレスを軽減します。
  • PC/スマホで引き継ぎ PiXiEEDrawファイルやPNGを書き出してGoogle Driveなどのクラウドに保存すれば、家のPCと外出先のスマホで同じ作品を続けて描けます。
  • リビルド不要のタイルプレビュー タイルセットを描いた瞬間に、同じパレットを使うシーン全体へライブ反映。配置済みマップを壊さず配色チェックができます。

よくある質問

リアルタイムでみんなで描けますか？
共有モード（マルチ）で、プロジェクトキーを使って複数人で同時に描画できます。マスター/参加者の役割とセル割当で制作できます（テスト運用中）。
アニメーション制作に対応していますか？
フレーム編集、FPS調整、オニオンスキン表示に対応しています。完成したデータはGIFとして書き出しできます。
PCとスマホで続きを描けますか？
PiXiEEDrawファイルやPNGを書き出してGoogle Driveなどのクラウドに保存すれば、家のPCで作業した続きがスマホでも開けます。
インストールは必要ですか？
ブラウザで動作するため、インストール不要でURLを開けばすぐに使えます。
保存はどうなりますか？
編集中はローカルに自動保存され、PNGやスプライトシート、タイルセットとして書き出しできます。
対応端末は？
PC・タブレット・スマホに対応し、Chromeでの利用を推奨しています。
