overview

概要

PiXiEELENSは、写真やカメラ映像をドット絵風の見た目に変換するためのツールです。

撮って終わりではなく、そのままPiXiEEDrawへつなげて仕上げる導線を持てます。

レトロ写真風の見た目をすぐ試したい人や、素材の下処理をしたい人に向いています。

features

主な特徴

カメラでも画像でも試せる

思いついた時にその場で見た目を変えられます。

ドット絵らしい粒度調整

粗さや雰囲気を確認しながら調整できます。

PiXiEEDrawへつながる

変換後の仕上げが別導線にならず、次の行動が明確です。

スマホ前提で見やすい

撮影から確認まで縦画面でも扱いやすくします。

scenes

利用シーン

  • 写真をレトロな見た目にしたい時
  • ドット絵素材の下絵を作りたい時
  • その場で見え方を試してSNSに使いたい時

how to use

使い方

  1. PiXiEELENSを開く
  2. カメラか画像を選ぶ
  3. ピクセル化の見え方を調整する
  4. 必要なら保存またはPiXiEEDrawへ送る

faq

FAQ

スマホでも使えますか？

使えます。カメラ利用時はブラウザの権限許可が必要です。

撮った後に編集できますか？

必要ならPiXiEEDrawへ送って仕上げできます。

登録は必要ですか？

不要です。ブラウザでそのまま使います。
